使用规则

一、平台承诺

零费用承诺
  • 平台 不收取任何中介费 / 服务费 / 信息费
  • 禁止以任何形式的收费或变相收费
  • 如发现违规收费，可一键举报
  • 经核实涉事账号将 永久封禁

二、租客使用指南

1

免费发布找房

点击 「立即找房」，填写您的租房需求

系统将根据 位置 + 时间 + 条件 自动匹配合适房源

匹配成功后，您可看到房源坐标及基础信息

2

获取联系方式（房豆机制）

  • 查看真实房东电话需消耗 1 房豆
  • 每日系统自动赠送 3 房豆
  • 房豆不足时可 观看广告免费获得
  • 所有找房需求发布 完全免费

三、房东使用指南

1

免费发布房源

点击 「立即出租」，填写房源信息并上传图片

系统会根据房源位置自动匹配合适租客

2

获取租客联系方式（房豆机制）

  • 查看租客联系方式同样需要消耗 1 房豆
  • 每日自动获取 3 房豆，可观看广告额外补充
  • 请确保房源信息真实有效，以免影响匹配效果

四、商家使用指南（如已开通）

  • 免费录入房源
  • 系统按位置与时间分发意向租客
  • 联系租客同样使用 房豆机制

五、安全提醒

  • 平台采用"房豆"验证机制以保护双方隐私
  • 建议线下看房时 尽量结伴同行
  • 不要提前支付任何形式的押金 / 订金 / 转账
  • 若遇到可疑行为，请立即向平台举报

六、关于匹配机制（透明说明）

系统根据以下维度进行智能匹配：

地理位置距离（附近优先）
发布时间先后（新发布优先）
租金 / 户型 / 面积等需求条件

匹配成功后将在地图上展示对方的房源位置或需求位置。
